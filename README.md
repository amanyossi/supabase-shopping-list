# 🛒 My Shopping List App

אפליקציית רשימת קניות חכמה המבוססת על **Vanilla JavaScript** ו-**Supabase**. האפליקציה מאפשרת ניהול פריטים בזמן אמת עם סנכרון מלא בין מכשירים.

## 🚀 תכונות (Features)
* **ניהול פריטים (CRUD):** הוספה, מחיקה ועדכון סטטוס (נקנה/לא נקנה) של פריטים.
* **סנכרון בזמן אמת (Realtime):** עדכון אוטומטי של הרשימה בכל הכרטיסיות הפתוחות ברגע שמתבצע שינוי בבסיס הנתונים.
* **אבטחת נתונים (RLS):** הגדרת Row Level Security המאפשרת גישה בטוחה לנתונים.
* **ממשק RTL:** עיצוב נקי המותאם לעברית.

## 🛠 טכנולוגיות (Tech Stack)
* **Frontend:** HTML5, CSS3, JavaScript ES6.
* **Backend:** Supabase (PostgreSQL & Realtime engine).
* **SDK:** `@supabase/supabase-js`.
